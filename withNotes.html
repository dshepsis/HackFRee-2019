Step 1: Vectors graphics

To be very brief, computers normally display things as a grid of squares called "pixels", each of which is assigned a color. This is called a "bitmap" image, and it's pretty great for a lot of things, like pictures, paintings, or video. One big problem, though, is that you can't use enlarge or zoom into a bitmap image without losing quality, because the bitmap image doesn't contain information about what's between the pixels. This causes the blocky, pixelated look on the right.

Unlike a bitmap, vector images are defined by math and geometry. Every shape is defined using a group of lines and curves which the computer then uses to calculate what colors are where in the image, sort of like a graphic calculator. And just like a graphing calculator, you can zoom in infinitely without losing any detail. This makes vector graphics great for simple images that you want to be sharp at any resolution.

Step 2: Making Vector Images

  • Standard file format: .SVG (Scalable Vector Graphics)
  • Text-file with XML-based formatting.
The most common file format for vector images is Scalable Vector Graphics, or .svg for short. It's a human-readable file-format, based on XML.

SVG Example

<svg width="500" height="500" viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
	<ellipse cx="50" cy="50" rx="40" ry="40"
		style="fill:orange;"
	/>
	<rect x="50" y="25" height="30" width="40"
		style="stroke:black; stroke-width: 10; fill: blue;"
	/>
</svg>
Since it's basically just code, you can write SVG files by hand if you want. That said, it's usually a lot easier to use a graphical editor, such as Inkscape.
This is what Inkscape looks like. If you're following along on your own machine, now is when you should have the software open. If you've ever used a photoediting software like Photoshop or GIMP, this may look familiar. If not, you might be feeling a little intimidated. Inkscape: Shapes

These are the 4 simplest, most basic tools in Inkscape. You select these form the left menu, or using one of a number of keyboard shortcuts that you can discover by hovering your mouse over the button.
****** Make sure you open Inkscape for real, right about now ******
The first tool, the cursor, can almost always be accessed by pressing Escape 1-3 times. It lets you select objects by clicking. Clicking once gives you the stretching handles. Clicking twice gives you the rotation and skew handles. You can also click and drag anywhere inside the dotted box to drag the object around.
There's a few important things here with the modifier keys: shift, control, and alt. When held, each of them has some effect on how Inkscape behaves. For example, if you hold CONTROL while dragging an object, it will only move horizontally or vertically, which is useful for maintaining alignment. If you hold control while stretching an object, it will maintain its aspect ratio. If you hold control while rotating or skewing an object, it will snap the angle to 15 degree increments. If you hold Shift, it will stretch the object in both directions. There's a lot more effects I won't go over here, but you can look them up, and the dialogue text at the bottom of the window will also give you hints at times.
Oh, and another quick note. You can change the what displays in front of or behind what by using the page up and down keys, or the home and end keys, while you have an object selected.

Inkscape: Paths

These are my favorite tools, and they're the most complicated thing I'll be talking about today, so pay attention. Paths are curves between 2 or more points (aka nodes). They can overlap and intersect themselves. The curves between nodes can be straight lines or curvy lines called bezier curves. The easiest way to make a path is with the pen tool. You click to start the path, then click for each point you want to add. If you click and drag, you'll make the last segment of the path curved. When you're done, you can either click on your starting point to close the path, or right click anywhere, which leaves the path open.
Once you have a path, you can edit it with the path tool. When you click on a path with this tool active, you'll see the nodes shown as squares or diamonds. You can click and drag nodes to move them, or click and drag the segments to change their curvature. If you click on a node next to a curved segment, you'll see circular handles appear, which are another way to adjust the curvature of adjacent segments.

Inkscape: Fill and Stroke

The fill-and-stroke menu lets you change the color or pattern of the inside or outline of a shape or path. If you don't already have it open, you can open it by pressing control+shift+f, or by going to the object menu at the top. This menu is pretty simple, but it's worth noting that even unclosed paths can have a fill. The fill area just behaves like the two ends are connected. Another useful tool is the stroke-style submenu, which lets you change width of the stroke, and make the stroke dashed, or add arrows to either end.

Inkscape: Boolean Operations

The Path menu has a bunch of very useful tools for combining paths in various ways. Union combines two paths. Difference takes the area of the top path and cuts it out of the bottom path. Intersection produces a new path which just contains the area covered by both paths. Exclusion is like difference, but the overlapping area is removed from both paths. The other options are more niche, and I'll leave them for you to research.
This menu also has some other nice tools, like the Object to Path button, which lets you turn shapes into paths, so you can edit them node-by-node, or use them in boolean operations. One of my favorites is Dynamic Offset, near the bottom. This removes all the handles on your path and gives you just one, which lets you expand or contract the perimeter of the path. I often use this to make gaps between parts of a logo, which I'll show you later.

Inkscape: Export PNG

While you can use SVG images in a lot of places, you'll often want to convert them into PNG images for one reason or another. Inkscape makes this relatively easy. To open the export menu, go to the file menu and click "Export PNG Image", or just press control+shift+E. Unfortunately there's no tool to just dragging over the area you want to export, so you'll usually want to either put a rectangle under the image and export selection, or change the size of the document and export Page.

Inkscape: Grid

The page grid is one of the most useful tools in Inkscape, because it lets you make repeatable diagrams with consistent spacing. To turn it on, go to the view menu at the top, or press shift+3, which gives the hash mark.

Inkscape: Snapping

To go with the grid, there is the snapping menu. Snapping causes objects, paths, nodes, and other things to snap to the grid or each other when moved around. This lets you place things precisely, without having to move them pixel-by-pixel. The snapping menu is usually located on the right side of the window, and each button is a toggle. I've labelled them all for you, but the important ones are bolded. In general, if you want something to snap but it isn't snapping, you probably need to play around with this menu. It's okay to not memorize the whole thing, because each button gives a tooltip when you hover over it.

Inkscape: Document Properties

You can open the Document Properties dialogue by going to the File menu at the top, or by pressing control+shift+D. This menu has a lot of important and niche options, but the ones I think are essential are the size menu, the background-color option, and the Grid menu. The size menu lets you set the size and units of the page, including pixels, milimeters, and inches. The background color menu is important because the default color of the page is actually transparent, which means your exported PNGs will come out with no background. Sometimes this is what you want, but you often want the background to be white, so this is where you would change that. The Grid menu lets you change the spacing and color of the grid. It also lets you make isometric grids, which are great for 3D technical drawing.
